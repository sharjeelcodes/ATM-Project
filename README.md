# ATM-Project
A simple Python-based ATM Management System built using modular programming. This project includes user authentication, balance inquiry, cash withdrawal, fast cash options, and mini statement features.  It is divided into multiple modules such as login, menu, and transaction handling to demonstrate clean code structure and separation of logic.
💡 Features:
PIN-based login system (3 attempts)
Check account balance
Cash withdrawal system
Fast cash options
Mini statement (last 5 transactions)
Modular Python structure
🛠 Technologies Used:
Python
File-based modular programming
📁 Project Structure:
login.py (Authentication)
withdraw.py (Transaction logic)
menu.py (User interface & control flow)
